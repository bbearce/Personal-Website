Projects

Brewculator

Personal Hombrew Recipe Builder



    This is a passion project of mine. A couple years ago my buddy Kyle Schluns and I started brewing our own beer recipes. We would use excel spreadsheets, and still sometimes do, to track our recipes and to calculate our ingredient quantities. I first built this very same app in R and it's 2 projects down(R Brew App) on this site. I then moved to Rmd documents, with all the calculations baked into the code. This is represented by the site in the below project (The Grateful Brauer). Each detailed recipe under the Ales and Lagers tab is an Rmd rendered page. This allows for a forum like post that we can use for doing a brain dump when researching a beer. The project on display here however, is my full stack solution. This is a Flask app composed of python, postgres, html, css, and javascript and utilizes AJAX calls for saving, loading, and deleting recipes. This is awesome, because it means we don't have to refresh the page and lose our progress when we save a recipe. It uses a couple of javascript packages for charts from chartjs and a little jQuery.

R Brew App

Design and Save Your Own Recipe



    When trying to design a beer recipe, there are many things that need to be considered and calculated. Previously my partner and I would use excel to do our calculations, but the equations are quite complex. We were using an excel template called Brew Cipher and it works quite well. The only problem was each recipe was a separate excel file, and if we wanted to customize any of our own math or add pieces to the equations, deciphering all the formulas was a challenge. We needed a solution that would allow us to save and load recipes and calculate exactly what we wanted. I transcribed all of the equations in Brew Cipher and tweaked what we needed to suit our needs. We now have a brewing application that can track and calculate what we need in order to make the beer we wish to make.
